# REPORT_20260318_JLeagueAttendance_FeatureEngineering.md

## 1-1. エグゼクティブサマリー (Summary)
- 予測精度向上に向けた特徴量エンジニアリングおよびデータ前処理を完了。
- 天候の簡略化、人気チームフラグ、ダービーマッチフラグなどの高次特徴量を生成。
- すべてのデータについて欠損値処理を行い、機械学習モデルに投入可能な状態とした。

## 1-2. 分析の目的と対象 (Scope)
- **目的:** 観客動員数予測モデルの精度向上に寄与する特徴量の作成。
- **対象データ:** `processed_data/merged_train.csv` (train + condition + stadium)

## 1-3. ドメインスペシャリストの見解 (Domain Insight)
- 天候は「雨が降ったか否か」が最も大きなバイアスとなるため、詳細な分類よりも「Rainy」かどうかの識別が重要。
- 人気チーム（浦和、横浜FM等）や、地理的・歴史的背景を持つ「ダービー」は、通常のカードとは異なる動員メカニズムが働く物理的実態がある。

## 1-4. 分析結果と考察 (Analysis & Findings)

### 1-4-1. 天候のカテゴリ化 (weather_cat)
- 30種類以上の複雑な天候情報を「Sunny」「Cloudy」「Rainy」「Indoor」の4種に統合。
- 集計結果:
  - Sunny: 942試合
  - Cloudy: 424試合
  - Rainy: 307試合
  - Indoor: 48試合
- モデルの複雑性を抑えつつ、降水による負の影響を捉えやすくした。

### 1-4-2. 人気チームフラグ (is_popular)
- 浦和レッズ、鹿島アントラーズ等の主要人気クラブを含む試合をフラグ化（計451試合）。
- これらの試合は平均動員数がベースラインより高い物理的傾向を持つ。

### 1-4-3. ダービーマッチフラグ (is_derby)
- さいたまダービー、大阪ダービー、多摩川クラシコ等の主要ダービーを定義（計25試合）。
- サンプル数は少ないものの、非常に強い集客力を持つため、個別変数として扱う価値が高い。

## 1-5. 結論と今後のアクション (Conclusion & Recommendations)
- **結論:** 前処理および特徴量生成が完了し、モデル構築フェーズへの準備が整った。
- **次の一手:** [ISSUE-03] において、これらの特徴量を用いたベースラインモデルの構築と要因分析を実施する。
