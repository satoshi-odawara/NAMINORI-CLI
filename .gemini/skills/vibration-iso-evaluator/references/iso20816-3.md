以下に、振動診断技師が現場でそのまま使えるレベルで、**ISO 20816-3（機械振動評価規格）**をマークダウン形式で体系的にまとめます。

---

# ISO 20816-3 詳細まとめ（振動診断技師向け）

## ■ 概要

**ISO 20816-3**は、回転機械の振動状態を評価するための国際規格であり、
特に以下の機械を対象とする。

* 電動機
* ポンプ
* ファン
* コンプレッサ
* タービン（小中型）

👉 主に**軸受ハウジング（ベアリング部）での振動速度（RMS）**を評価指標とする。

---

## ■ 評価の基本指標

### ● 振動速度（Velocity RMS）

* 単位：mm/s（rms）
* 周波数範囲：**10 Hz ～ 1000 Hz**

👉 理由

* 低周波（アンバランス・ミスアライメント）
* 中周波（緩み・共振）
  をバランス良く検出できるため

---

## ■ 機械分類（クラス）

ISO 20816-3では機械を剛性・設置条件で分類する

| クラス       | 内容               |
| --------- | ---------------- |
| Class I   | 小型機械（15 kW未満など）  |
| Class II  | 中型機械（15〜75 kW程度） |
| Class III | 大型機械（剛性基礎）       |
| Class IV  | 大型機械（柔らかい基礎）     |

---

## ■ 評価ゾーン（状態判定）

振動レベルに応じて4段階評価

| ゾーン    | 状態 | 意味           |
| ------ | -- | ------------ |
| Zone A | 良好 | 新品または非常に良い状態 |
| Zone B | 許容 | 長期運転可能       |
| Zone C | 注意 | 継続運転可能だが対策必要 |
| Zone D | 異常 | 即時対応必要       |

---

## ■ 判定基準値（代表例）

### ● Class II（中型機械）の例

| ゾーン | 振動速度（mm/s rms） |
| --- | -------------- |
| A   | 0 ～ 1.12       |
| B   | 1.12 ～ 2.8     |
| C   | 2.8 ～ 7.1      |
| D   | 7.1 以上         |

---

### ● Class III（大型・剛基礎）

| ゾーン | 振動速度（mm/s rms） |
| --- | -------------- |
| A   | 0 ～ 1.8        |
| B   | 1.8 ～ 4.5      |
| C   | 4.5 ～ 11.2     |
| D   | 11.2 以上        |

---

## ■ 測定方法

### ● 測定位置

* 軸受ハウジング（水平・垂直・軸方向）

### ● 測定方向

* H（水平）
* V（垂直）
* A（軸方向）

👉 最も大きい値を評価に使用

---

### ● 測定条件

* 定常運転状態
* 定格回転数付近
* 外乱が少ない状態

---

## ■ 評価の考え方（重要）

### ① 絶対値評価

→ 上記ゾーンで判断

---

### ② トレンド管理（実務で最重要）

* 急激な増加 = 異常兆候
* 基準値以下でも注意

👉 例

* 2.0 → 3.0 mm/s（まだBだが異常進行）

---

## ■ 典型的な異常と振動特徴

| 異常       | 特徴         |
| -------- | ---------- |
| アンバランス   | 1×回転数成分増大  |
| ミスアライメント | 2×成分・軸方向増大 |
| 軸受損傷     | 高周波成分増加    |
| 緩み       | 広帯域増加・衝撃   |
| 共振       | 特定周波数で急増   |

---

## ■ 注意点（現場で重要）

### ● ISOはあくまで目安

* 機械固有特性に依存
* 全機械に完全適用は不可

---

### ● 設置条件の影響

* 柔基礎 → 振動大きくなりやすい
* 剛基礎 → 振動小さくなる

---

### ● 周波数解析と併用必須

👉 ISOだけでは原因特定不可

---

## ■ ISO 10816との違い

| 項目 | ISO 10816 | ISO 20816 |
| -- | --------- | --------- |
| 状態 | 旧規格       | 新規格       |
| 内容 | 基本評価      | より細分化     |
| 更新 | 廃止方向      | 現行標準      |

---

## ■ 実務での使い方（まとめ）

### ✔ 基本フロー

1. 振動速度RMS測定
2. クラス判定
3. ゾーン判定
4. トレンド確認
5. FFT解析で原因特定

---

## ■ 現場での判断基準（実践）

* **Zone A〜B** → 問題なし
* **Zone B→Cに上昇** → 要監視
* **Zone C** → 計画保全
* **Zone D** → 即停止検討

---

## ■ 補足（重要ポイント）

👉 ISO 20816-3は
**「異常の有無」ではなく「状態レベル」評価規格**

👉 診断には必ず

* FFT
* 包絡線解析
* トレンド

を併用すること

---

# ■ まとめ

ISO 20816-3は以下を提供する規格：

* 振動速度による機械状態の定量評価
* 客観的な保全判断基準
* トレンド管理の基準

しかし、

👉 **原因診断は別途解析が必須**
