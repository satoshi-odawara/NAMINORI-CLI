# Knowledge

- サイドバンド解析:
  - 内輪欠陥 (BPFI): 基本欠陥周波数の周囲に $1 \times$ RPM 間隔のサイドバンドが現れる 。
  - ギヤ摩耗: 噛み合い周波数 (GMF) の周囲に回転周波数のサイドバンドが現れる 。
  
- 位相による識別:
  - アンバランス: 半径方向で同位相（安定） 。
  - ミスアライメント: カップリング両端で $180^\circ$（逆位相）の差が顕著 。


# Instruction

1. 抽出されたピークが回転数の整数倍か、ベアリング欠陥周波数 $\pm 3\%$ 以内かを確認してください 。
2. BPFI成分が検出された場合、周囲に回転数間隔のサイドバンドが存在するかを確認してください。
3. 半径方向と軸方向の振幅比を確認し、軸方向が高い場合はミスアライメントを疑い、位相データの照合を行ってください 。
