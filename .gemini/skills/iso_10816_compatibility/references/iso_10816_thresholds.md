# Reference: ISO 10816 Compatibility

ISO 10816 は旧規格ですが、一部の設備管理基準で依然として「変位ピーク値 [µm]」が使用されています。

## 変位評価閾値 (例: 中型回転機械)

| Zone | 変位ピーク-ピーク [µm] | 評価 |
|---|---|---|
| Zone A | < 25 | 良好 |
| Zone B | 25 - 50 | 許容 |
| Zone C | 50 - 80 | 注意 |
| Zone D | > 80 | 危険 |

*Note: 実際には設備回転数により閾値が変動しますが、本スキルでは汎用的な代表値をプリセットします。*

## 速度→変位変換式
変位 $d(t)$ は速度 $v(t)$ の積分として求められます。
$$d(t) = \int v(t) dt$$
周波数領域では、
$$D(f) = \frac{V(f)}{j 2\pi f}$$
として表現されます。
