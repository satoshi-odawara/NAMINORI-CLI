# Knowledge

- 実効値 (RMS): 全体的な摩耗エネルギー。
- 尖度 (Kurtosis): $3$ を超えると、軸受やギヤの初期ピッチング（衝撃）の可能性が高い。
- クレストファクタ: $4$ 以上は軸受の初期損傷を示すインジケータ。


# Instructions

- 時間波形から RMS、尖度 (Kurtosis)、歪度、CF を算出し、ガウス分布からの逸脱を評価してください。
- 尖度または歪度が上昇しているが RMS が正常な場合、「極初期の欠陥」として報告してください 。
- RMS が上昇している局面で CF が低下し始めた場合、それは健全化ではなく「欠陥の深刻化（全体的な崩壊）」と解釈し、至急の対応を促してください 。
