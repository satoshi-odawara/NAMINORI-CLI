# Knowledge

- MFCC: 振動データを 60〜100 倍に圧縮しつつ、衝撃や摩擦のフォルマントを抽出する 。
- GARCH(1,1): ボラティリティの持続性を評価。安定条件は $\alpha + \beta < 1$ である 。
- HRV (心拍変動) 解析の転用: 衝撃パルスの時間間隔（RR間隔に相当）の標準偏差 (SDNN) から、機械的な「リズム」の安定性を評価する 。


# Instructions

- DAS 等の大容量データでは MFCC 変換による圧縮を優先してください 。
- GARCHモデル推定時、$\alpha + \beta$ が $1$ に近い場合は「劣化が持続的かつ拡大傾向にある」と解釈してください 。
- ベアリングの衝撃パルスを QRS 検出アルゴリズムで抽出し、パルス間隔のゆらぎ (SDNN) が増大している場合は、ガタつきの進展として報告してください 。
