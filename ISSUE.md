- **担当ロール**: All
- **作成日**: 2026-03-27
- **完了日**: 2026-03-27

### 概要

ヒルベルト変換を用いて振動信号の包絡線を抽出し、そのスペクトルを分析するスキル。ベアリングの局所的な欠陥（転動体・内輪・外輪）は高周波共振を変調させるため、低周波のエンベロープスペクトルに欠陥周波数（BPFI、BPFO、BSF、FTF）として現れる。ベアリング診断の定番手法。

### 受け入れ条件（Definition of Done）

- [x] `scripts/run_envelope.py` が実装されている
- [x] 共振帯域のバンドパスフィルタ（中心周波数・帯域幅）をパラメータ指定できる
- [x] 出力 JSON に `envelope_spectrum`, `bpfi`, `bpfo`, `bsf`, `ftf`, `defect_frequencies` が含まれる
- [x] 標準 JSON 入出力スキーマ（§3-1）に準拠している

- [x] QA によるコードレビューが完了している

### 実装メモ

- 諸元から計算した理論欠陥周波数（BPFO等）との照合ロジックを組み込み、5%以内の誤差で自動アラートを出すようにした。
- CWRUデータセット等の実データ検証は、将来的に `assets/reference/` で実施予定。
