# データ分析依頼ヒアリングシート (v1.1)

## 1. 分析の目的・背景 (Goal)

- **解決したい課題:** 機械状態監視のための異常音の非監視型検出
- **期待する結果:** 正常な音のサンプルのみがトレーニングデータとして提供されている状況下で、未知の異常音を検出する.実際の工場では、異常音はめったに発生せず、非常に多様です。そのため、異常音の網羅的なパターンを意図的に作成したり収集したりすることは不可能です。つまり、与えられたトレーニングデータでは観測されなかった未知の異常音を検出する必要があるということです。trainフォルダのデータでトレーニングをおこない、testフォルダのデータでモデルの正しさを検証してください。
- **アウトプットの活用イメージ:** 予兆保全、異常原因の推測による、保全業務の効率化

## 2. データに関する情報 (Data Metadata)

- **対象ファイル名** train/test
- **データの内容・構造:** - 内容: このタスクで使用されるデータは、実機の正常/異常動作音を収録したMIMIIデータセットで構成されています。各録音は、対象機械の動作音と環境ノイズの両方を含む、シングルチャンネル（約）10秒の音声です。今回の対象機会はポンプです。ファイル名が正常(normal)と異常(abnomal)で分けられています。
  - 書式: wav/ヘッダーなし
  - カラム定義: なし
- **データの緒言:**
  - サンプリング周波数: wavファイル参照
  - 測定期間・量: 不定期
